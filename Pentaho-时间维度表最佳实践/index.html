# 目标表

  • 版本号：每次更新的将增加 1
  • 起始时间：数据起始的有效时间
  • 截止时间：数据截止的有效时间
  • 日期维 ID：自增 ID
  • 日期：年 - 月 - 日格式的日期
  • 年：年份
  • 月：本年的月
  • 日：本月的天
  • 季度：本月的季度

版本号，起始时间与截止时间用于区分最新的的数据与历史记录
目标表将分为两种抽取方式，分别是全量抽取与增量抽取

# 全量抽取

一次性把所需要的时间维度抽到维度表里

  1. 打开 pentaho data integration 目录下的 spoon.bat
  2. 新建一个转换

  4. 双击转换里的生成记录，并设置限制为 1，点击确定
  5. 搜索计算器，并拖到转换里。按住 shite 点一下生成记录，再点一下计算器把他们连起来

  8. 双击克隆行并配置如下

  10. 计算器 2 设置如下
  11. 然后输出到维度表
  12. 双击维度表，先新建一个数据库连接
  13. 配置好数据库连接的信息，测试通过后保存连接
  14. 配置目标表，维字段，代理关键字段，version 字段
  15. 配置好字段

    因为时间维度表一般是不需要保留历史记录的，所以不会做成缓慢增长维，因此这里选择更新而不是插入
  16. 保存运行后就能看见已经生成了 2018-01-01 到 2020-01-01 的时间维度了
