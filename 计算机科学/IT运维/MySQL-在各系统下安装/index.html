MySQL 是一个功能齐全的关系数据库管理系统（RDBMS），可以与 Oracle DB 和 Microsoft 的 SQL Server 竞争。MySQL 由瑞典公司 MySQL AB 赞助，该公司由 Oracle 公司拥有。

# Windows

  1. 下载 mysql，点击进入下载
    第一个是在线下载版，第二个是安装版
    我选择第二个
  2. 选择 No thanks 跳过登录
  3. 下载完成后打开安装包
    根据需求选择，这里选择 develop
  4. next
  5. 安装
  6. 安装完成
  7. next
  8. 选择高可用，这里选择单机模式
  9. 选择 mysql 端口，这里选择默认的 3306
  10. 选择密码认证方式，这里选择默认的第一个选择
  11. 输入数据库管理员密码
  12. 是否开机启动
  13. 开始自动 mysql
  14. 完成配置
  15. next
  16. 是否启动 boostrap，这里直接 next
  17. 连接数据库，输入之前输入的密码，点击 check
  18. 执行
  19. 完成
  20. 下一步
  21. 将 mysql 添加到开始菜单
  22. 点击开始菜单，打开 mysql 客户端
  23. 输入密码，若连接成功则安装完成

  1. 安装 mysql repo 源
  1. 安装 Mysql
  1. 运行 Mysql 服务，第一次运行
  1. 重置密码
$ mysql -u root
>use mysql;
>update user set password=password("123456") where user="root";
>exit;
